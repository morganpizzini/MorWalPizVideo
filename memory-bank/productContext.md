# Product Context - MorWalPizVideo

## Problems Solved
- **Video Organization** - Managing large YouTube collections with hierarchical structure (root + sub-videos)
- **Translation Bottleneck** - Automated batch translation vs manual effort
- **Multi-Platform Distribution** - Centralized short links and social media management
- **Content Discovery** - Categorization and searchability across video libraries

## User Personas
- **Content Creators** - YouTubers managing 10K+ videos, need translation and organization
- **Content Managers** - Admins handling social media distribution, short links, analytics  
- **Shop Customers** - End users purchasing digital products

## Core Workflows
1. **Video Import** - YouTube ID → fetch metadata → categorize → create collections
2. **Translation** - Select videos → Azure Translator → store translated metadata
3. **Social Distribution** - Create short links → distribute to Discord/Telegram → track usage
4. **Product Purchase** - Browse catalog → add to cart → checkout → digital delivery

## Value Delivered
- 90% reduction in translation time via automation
- Hierarchical video organization with root/sub-video model
- Single platform for YouTube + social + shop management
- Desktop bulk operations + web management interface
